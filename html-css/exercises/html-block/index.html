Block

Block-level elements always start on a new line and take up the full width available (stretching left to right as far as possible).BLOCK BLOCK Examples include <div>, <nav>, <header>, <section>, <footer> and <ul> elements to name a few.

Inline

Inline elements do not start on a new line and only occupy the space that their content requires.INLINE Some examples of inline elements include <span>, <input>, <label>, <select> and <a> elements.

Inline-block

Inline-block is set by the CSS display property, and differs to inline elements as we can set width and height. We cannot do this with inline elements as they only take the space their content needs. Also, top and bottom margins/paddings are respected, but with inline elements they are not.INLINE-BLOCK INLINE-BLOCK Unlike block-level elements, inline-block elements do not start on a new line and will sit next to other elements.
